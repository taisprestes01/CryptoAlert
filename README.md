## Key Features 

* Price Alerts:
  - Users can set up alerts to be notified when the price of a specific cryptocurrency reaches a desired buy or sell value.

* Real-Time Tracking:
  - Allows users to view the current value of their cryptocurrency portfolios on Binance, with automatic real-time updates.

* Trending Coins:
  - Provides information about cryptocurrencies that are trending, even if the user does not currently hold them in their portfolio, enabling them to explore new investment opportunities.

* Top Coins in a Specific Period:
  - Displays the top cryptocurrencies based on their performance during a specific period of time, helping users identify trends and trading opportunities.

* Worst Performing Coins in a Specific Period:
  - Offers a list of cryptocurrencies with the worst performance during a defined period of time, assisting users in avoiding high-risk investments.

* Highest Price Variation in a Period:
  - Shows the cryptocurrencies with the highest price variation during a specific period of time, helping users identify profit opportunities through market volatility.

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/amitmerchant1990/electron-markdownify

# Go into the repository
$ cd electron-markdownify

# Install dependencies
$ npm install

# Run the app
$ npm start
```

> **Note**
> If you're using Linux Bash for Windows, [see this guide](https://www.howtogeek.com/261575/how-to-run-graphical-linux-desktop-applications-from-windows-10s-bash-shell/) or use `node` from the command prompt.
